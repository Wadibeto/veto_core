# Troubles urinaires (focus chat male)

## Signes frequents
- Efforts repetes pour uriner.
- Mictions tres petites ou absentes.
- Douleur, vocalisation, lechage genital, sang dans urines.

## Point critique
Chez le chat male, l'absence d'urine malgre efforts est une urgence vitale potentielle.

## Questions utiles
- Date de la derniere miction normale?
- Quantite d'urine observee?
- Douleur manifeste? Sang?
- Antecedents urinaires connus?

## Conduite
- Ne pas attendre a domicile si suspicion d'obstruction.
- Consulter en urgence immediate.

## Message de securite
Le delai peut aggraver rapidement l'etat general.
